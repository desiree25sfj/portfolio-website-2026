Projects

Three C# projects that show where I am heading.

These C# projects are not pretending to be enterprise systems. They are better than that for a junior portfolio: they show language fundamentals, backend thinking, and the ability to build through confusion instead of stopping at the first weird error.

Project spotlight

A playable web adaptation of my C# WinForms Tetris project.

The original project is a C# Windows Forms app. This browser version mirrors its core behavior for the portfolio: seven tetrominoes, arrow-key movement, collision detection, line clearing, fixed fall speed, and the same line-clear scoring. It also adds a small wall-kick improvement so the demo feels smoother.
